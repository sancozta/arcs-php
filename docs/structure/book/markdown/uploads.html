Uploads

Uploads

Os arquivos que são enviados por meio do upload de arquivos são colocados no diretório /assets/files, com isso refoorçamos o cuidado com esse diretório. Ele não esta sendo excluído do controle de versão pela decisão de manter a salvo os arquivos que são enviados para o servidor. Nesse diretório podem existir arquivos com até 10MB que é o limite permitido pela aplicação.

Nota !

Por favor manténha os arquivos sempre atualizados com a versão do sistema. Sempre documente os objetos criados e caso ocorra alteração especifique a nova função do elemento criado. Os arquivos contidos nesse diretório devem ser uma cópia fiel dos arquivos sendo utilizados pela aplicação, com o objetivo de facilitar o entendimento de novos desenvolvedores da situação atual do projeto.
